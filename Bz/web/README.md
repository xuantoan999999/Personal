
# Introduction

Main Application

## Requirements

- Nodejs 6.0 and above
- MongoDB
- Redis
- Nodemon: `npm install -g nodemon`

## Install

```bash

npm install
bower install

```

## How to run:

```bash
yarn start
```
Portal: `http://localhost:9009/`
Website: `http://localhost:9006/`
API: `http://localhost:9001/`
CMS: `http://localhost:9000/`
CMS Account: admin@gmail.com/iii3studi1
